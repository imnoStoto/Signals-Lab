# Signals-Lab
Early experiments with digital signal processing.

## Overview

This repository is a signal-processing lab spanning **audio DSP** and **biomedical DSP**.  
The biomedical track focuses on **reproducible, pipeline-first handling** of WFDB/PhysioNet datasets (e.g., MIT-BIH Arrhythmia).

A core principle: **raw datasets are immutable**. All downstream artifacts are derived and reproducible.

---

## Repository Data Zones

- `dataset/raw/...`  
  Immutable raw dataset files (e.g., `.hea`, `.dat`, `.atr`).  
  **Never modified. Never committed to git.**

- `dataset/clean/...`  
  Derived metadata and quality artifacts generated by scripts:
  - `manifest.csv` / `manifest.parquet` (inventory)
  - `record_quality.csv` (validation results)
  - `usable_records.csv` (approved subset for DSP/features)
  - `validation_summary.json` (counts + thresholds)
